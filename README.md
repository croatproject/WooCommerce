# WooCommerce

* Multilingual (WPML)
* Multilingual (Polylang) - Need WooPolyIntegration (https://github.com/hyyan/woo-poly-integration/releases)
* Set fixed parity
* Use market parity - Updated from (https://www.worldcoinindex.com)
