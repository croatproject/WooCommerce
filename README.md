# WooCommerce
WooCommerce Croat plugin

* Multilingual (WPML)
* Multilingual (Polylang) - Need WooPolyIntegration (https://github.com/hyyan/woo-poly-integration/releases)
* Set fixed parity
* Set parity source
* Set number of decimals
* Show / Hide price with CROAT
* Show / Hide QR Code
* Show / Hide Wallet Address
